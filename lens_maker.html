Optik Laboratuvarı: Mercek Yapıcı Denklemi

Denklem:
1/f = (n - 1) * (1/R1 - 1/R2)

build Mercek Tasarımı

Kırılma İndisi (n): 1.50

Mercek yapımında kullanılan malzemenin ışığı ne kadar kıracağının bir ölçüsü (Hava = 1, Cam ≈ 1.5, Elmas ≈ 2.4).

1. Yüzey Yarıçapı (R1): 200 birim

Sol yüzeyin eğrilik yarıçapı. R1 > 0 ise yüzey ışığa göre tümsek, R1 < 0 ise çukurdur.

2. Yüzey Yarıçapı (R2): -200 birim

Sağ yüzeyin eğrilik yarıçapı. R2 < 0 ise yüzey ışığa göre tümsek, R2 > 0 ise çukurdur.

Hesaplanan Odak Uzaklığı

1/R1 - 1/R2 : 0.010
(n - 1) : 0.50
Odak Uzaklığı (f): 200.0
